auth Module

Defined in: lib/components/auth.js:36

Basic authentication mechanism.

The Auth Component currently does basically a few things:

When initialized, it will lookup at the settings - src/config/main.js - and will listen to any request made to the selected login route - config.loginUrl -. Afterwards, will instantiate the selected model - config.model - and will use model's 'auth' method to execute custom authentication logic (db lookup, etc). Internally, a cookie is generated - named 'auth' - containing basic user information.

The configuration parameters are as follow:

loginUrl - URL receiving POST fields for user information (username, password).
